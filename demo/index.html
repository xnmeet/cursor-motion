Built for realism.

Every detail tuned to produce motion that feels genuinely human.

Spring Physics

Velocity-Verlet integration at 240Hz. Underdamped springs produce natural overshoot and settle, just like real physical systems.

7-Path Intelligence

Generates 7 candidate bezier paths per move — direct, turn, brake, orbit — then scores each on length, curvature, and context.

Zero Allocation

Hot path uses flat scalars and pre-allocated outputs. No GC pressure, no frame drops — even at sustained 60fps.

Visual Dynamics

Independent tip spring + angle spring create realistic cursor tilt and lag. The pointer rotates into its heading naturally.

Dirty-Check DOM

Renderer skips identical transforms. Only writes to DOM when values actually change — no forced reflows, no layout thrash.

TypeScript Native

Written in strict TypeScript with full type exports. Every config, state, and output is properly typed for IDE intelligence.

Three lines to natural motion.

Import the engine, set your target, and let physics handle the rest. Full control over spring parameters when you need it.

  • Auto path selection from 7 candidates
  • Configurable spring response & damping
  • Built-in DOM renderer with fog effect
  • Chain multiple moves seamlessly
  • Full event callbacks (start, move, end)
TypeScript import { CursorMotion } from 'cursor-motion-js' // Initialize with spring config const cursor = new CursorMotion({ response: 1.4, damping: 0.9, container: document.querySelector('.app') }) // Move to target — physics handles the path cursor.moveTo({ x: 420, y: 280 }) // Chain moves with custom options await cursor.moveTo( { x: 800, y: 150 }, { arcSize: 0.12, arcFlow: 0.7 } )

How it works

Physics simulation in three stages.

1

Generate Candidates

7 bezier paths are generated based on direction, heading, and distance — direct, turn, brake, and orbit families.

2

Score & Select

Each path is measured for length, curvature energy, and turn demand. The lowest-score path wins — naturally favoring smooth motion.

3

Spring Animate

A Velocity-Verlet spring drives progress along the chosen path at 240Hz, while tip and angle springs add realistic dynamics.
